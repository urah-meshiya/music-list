# 歌リクくん

## ■ 概要
 * Googleスプレッドシートから読み取ったデータを元に、PC・スマホ(iPhone/Android)・タブレットいずれにも対応した形式で楽曲のリスト（以下「リスト」）を表示します。
 * スプレッドシートの内容を変更すると、自動的にリストにも反映されます。

## ■ 提供いただくもの
 * Googleスプレッドシートで作成したデータ
   * ご自身のGoogleDriveからの共有が難しい場合は当方でシートをご用意することも可能です。
   * テンプレートは[こちら](https://docs.google.com/spreadsheets/d/11IxSIguxpyZIWu-kUb7nztYC2aBKgYuNk5-ZPrOAALM)。コピーして利用いただけます。
   * シートは「リンクを知っている人全員」に共有してください。
   * ヘッダー行（各列の説明を書く行）を最初に設けることをオススメします。
   * 列の内容は曲名以外は任意です。
     * オススメは、「作者名(ボーカル名)」「曲名の読み」「作者名(ボーカル名の読み)」、「熟練度(オススメ度)」、「年代」など。
     * リストに表示させる列は自由に選択できます。<br>「曲名の読み」など並び替えのためだけに必要な列は、シート外では非表示にするのがオススメ。
 * アイコン(任意)：透過画像推奨
 * 背景用画像(任意) 
 * favicon用画像(任意)

## ■ 機能紹介
### 基本機能
 * ユーザ別HTML提供
   * デザイン（背景、色、画像、フォント、テーブル幅 など）はユーザ毎にカスタマイズ可能
 * Googleスプレッドシート読み取り機能
 * リストのテーブル表示機能

### カスタム機能
 * リストの並び順切り替え機能
 * 検索/絞り込み機能
   * 部分一致検索 または 前方一致検索 が選択可
 * インフォメーション機能
   * 曲名の右横にボタンを表示、クリックorタップ時に情報を表示します。歌唱キーやスケールのメモに便利です。
 * 読込中の画像表示機能
 * ランダム選曲機能
   * 特定の条件（例: URL入力済みのみ、高熟練度のみ）での選曲にも変更可
 * 流れ星機能
   * 任意のボタンや画像をクリックした際、任意の画像を画面上に降らせます
 * リクエスト機能
   * リクエストボタン押下時、bot（[こいつ](https://ja.twitcasting.tv/c:sing_request)）が該当ユーザの配信にコメントを投稿する
   * 配信中でない場合は機能しない
   * 同一IPからの連投制限（荒らし防止。デフォルトは10分当たり3件。ユーザ毎に変更可）
   * リクエスト履歴は別途URLから確認可（リストと同ページに表示も可）
 * その他
   * 応相談。色々できます。ご相談ください。

## ■ サービス比較
歌リクくん(これ)と同様に楽曲のリスト管理に使われるサービスとの比較
|  | メリット | デメリット |
| ---- | ---- | ---- |
| 歌リクくん | * 好き放題デザインできる <br> * メンテナンスも樂 <br> * 曲数が多くても対応可 <br> * 共有編集可能 <br> * 特有のリクエスト機能あり | * 作るのが大変(主に俺が) <br> * 歌唱リクエストは匿名投稿 |
| [mimemo](https://mimemo.io/) | * 作るのが簡単<br> * 共有編集可能 <br> * 曲数が少ないならこれで十分 | * 単なるテキストのため並び替え操作やデザインは不可 | 
| [lit.link](https://lit.link/) | * 作るのが簡単<br> * リンク集を兼ねられる <br> * 曲数が少ないならこれで十分  | * 単なるテキストのため並び替え操作やデザインは不可 <br> * 管理が大変（本来リスティングを目的としたサービスではない） | 
| [setlink](https://setlink.jp/lp) | * 高機能(楽曲DB化、セトリ作成 ほか)で有償版もある | * 細かいデザインは不可 <br> * カネの臭いがする(私怨) |
| [notion](https://www.notion.com/ja) | * 高機能 | * 慣れないと操作が難しい<br> * デザインも最低限 |
| [Googleスプレッドシート(直共有)](https://support.google.com/docs/answer/6000292?hl=ja) | * 色々できる(関数、フィルタ、クエリほか)<br> * 共有編集可能 | * 慣れないと操作が難しい<br> * デザインも不可 |
